# Bachelor Thesis Code

This repository contains the Python scripts used for the analyses reported in my bachelor thesis on adversarial attacks and adversarial training of deception classifiers.

## Files:
- **comparison.py** \
  Evaluates and compares the efficiency and effectiveness of human- and LLM-generated adversarial paraphrases against the deception classifier.
- **adversarial_training.py** \
  Performs adversarial training using human-generated, LLM-generated, and combined paraphrases, and evaluates their impact on classifier performance and robustness.


The datasets and pretrained base model are not included because they were provided as part of the project and are not publicly distributed by the author.
